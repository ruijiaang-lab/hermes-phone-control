# 推特草稿全链路

从手机微信一条指令，完成推文润色→本地备份→存入 X 草稿箱。

## 流程

```
微信发消息: "今天打通了链路，躺着就把推发了。存"
    │
    ▼
Hermes Agent
    ├── 1. 润色（按账号人设）
    ├── 2. 本地备份 → ~/Desktop/推特草稿/标题.md
    └── 3. Playwright → X 草稿箱
    │
    ▼
微信回复: 润色结果 + 保存状态
```

## 前置条件

```bash
# 安装 Playwright
pip3 install playwright
python3 -m playwright install chromium

# 准备浏览器 profile（首次需要手动登录 X）
mkdir -p ~/x-draft-profile
```

## 脚本

`x_save_draft.py` — 用 Playwright 把内容存入 X 草稿箱：

```python
#!/usr/bin/env python3
"""Save a tweet draft to X via Playwright persistent context."""
import sys, os, time
from playwright.sync_api import sync_playwright

PROFILE_DIR = os.path.expanduser("~/x-draft-profile")
PROXY = "http://127.0.0.1:7897"  # Clash Verge 或其他代理

def save_draft(content: str):
    # 清理锁文件
    for f in ["SingletonLock", "SingletonSocket", "SingletonCookie"]:
        p = os.path.join(PROFILE_DIR, f)
        if os.path.exists(p):
            os.remove(p)

    with sync_playwright() as p:
        ctx = p.chromium.launch_persistent_context(
            PROFILE_DIR,
            headless=False,
            proxy={"server": PROXY},
            args=["--no-first-run"],
        )
        page = ctx.pages[0] if ctx.pages else ctx.new_page()
        page.goto("https://x.com/home")
        page.wait_for_load_state("networkidle")
        time.sleep(2)

        # 移除遮罩层
        page.evaluate('document.querySelectorAll("[data-testid=mask]").forEach(e=>e.remove())')

        # 按 n 打开发帖框
        page.keyboard.press("n")
        time.sleep(1)

        # 输入内容
        editor = page.locator('[data-testid="tweetTextarea_0"]')
        editor.click()
        editor.fill(content)
        time.sleep(1)

        # 点 X 关闭（不是 Escape！Escape 直接丢弃）
        page.locator('[data-testid="app-bar-close"]').click()
        time.sleep(1)

        # 点"保存"确认
        page.locator('[data-testid="confirmationSheetConfirm"]').click()
        time.sleep(2)

        ctx.close()
    print("✅ 草稿已保存到 X")

if __name__ == "__main__":
    if len(sys.argv) < 2:
        print("用法: python3 x_save_draft.py \"你的推文内容\"")
        sys.exit(1)
    save_draft(sys.argv[1])
```

## 关键 Selector

| 元素 | data-testid |
|------|-------------|
| 发帖按钮 | `SideNav_NewTweet_Button` |
| 编辑器 | `tweetTextarea_0` |
| X 关闭按钮 | `app-bar-close` |
| 保存草稿 | `confirmationSheetConfirm` |
| 放弃草稿 | `confirmationSheetCancel` |
| 遮罩层 | `mask` |

## 常见坑

1. **Escape ≠ 保存** — Escape 直接丢弃草稿，必须点 `app-bar-close`
2. **遮罩层** — X 有隐形遮罩层拦截点击，必须先用 JS 移除
3. **Profile 锁** — Chrome 运行时会锁 profile，Playwright 用独立目录
4. **代理** — Playwright 不走系统代理，必须显式配置
